Раньше было много HTML, мало - JavaScript.
Сейчас много JavaScript, мало HTML. 
Клиент постоянно делают AJAX(аякс) запросы к серверу. 
Сервер возращает данные JSON_(джейсон, ДЖЕ-ЭС-ОН).
Страница не перезагружается.
Изменения приходят с сервера и клиент сам дорисовывает HTML и всё остальное для того места, которое изменилось(например, пришло новое сообщениеVV);

При переходе на разные вкладки(/feed ----> /messages ----> /audio ) JavaScript перехватывает изменение url-a, делает нужный AJAX(аякс-запрос) на Сервер, клиент получает нужный JSON (джейсон, ДЖЕ-ЭС-ОН), странца перезатирается(/feed ----> /messages ----> /audio ).
Трафик уменьшается, скорость работы возрастает.

Single-page-application - это web-приложение, тут очень много логики. 

Формат JSON - универсальный. Этот формат формат считается независимым от языка и может использоваться практически с любым языком программирования. Для многих языков существует готовый код для создания и обработки данных в формате JSON.
